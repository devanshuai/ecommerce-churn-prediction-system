# 📊 E-commerce Customer Churn Prediction System

## 🚀 Overview
This project predicts customer churn in an e-commerce platform using machine learning.  
It provides risk segmentation, personalized recommendations, and an interactive dashboard for business insights.

---

## 🎯 Features
- 🔍 Churn prediction using ML model
- 📈 Risk segmentation (High / Medium / Low)
- 🧠 Customer-level churn reasoning
- 🎯 Personalized recommendations
- 📊 Interactive Streamlit dashboard
- 📥 Download reports (CSV & Excel)

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit
